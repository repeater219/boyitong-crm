# 生产实习小结（自我鉴定）

在安阳博易通网络有限公司为期十周的生产实习中，我参与了"博易通客户数据管理系统"项目的设计与开发工作。通过本次实习，我将所学的计算机专业知识应用到真实业务场景中，完成了从需求分析、系统设计到开发测试、部署上线的完整项目流程。

在需求分析阶段，我深入了解了公司销售团队在客户管理中的实际痛点——大量客户信息分散在 Excel 表格中，查询不便、权限混乱、协作效率低。在此基础上明确了系统的功能需求和技术方案。

在系统设计阶段，我采用了 Spring Boot 3.x + Spring Data JPA 作为后端框架，React + Ant Design 作为前端技术栈，使用 PostgreSQL 作为数据库，前后端分离架构，RESTful API 通信。这套技术方案符合当前主流的 Web 开发标准。

在开发阶段，我完成了客户信息管理、Excel 批量导入导出、多条件筛选与搜索、数据统计与可视化、用户认证与权限控制、操作日志、通知系统等基础功能，以及商机管理、合同管理、产品管理、跟进记录、公告中心等 CRM 核心模块。此外还接入了 AI 助手，实现了智能对话和跟进记录自动生成。前后端代码总计约 6200 行。

在测试与部署阶段，我编写了 95 个后端单元测试用例，覆盖了所有核心功能，测试通过率 100%。最终将系统通过 Docker 容器化部署到 Railway 平台，实现了线上可访问。

通过本次实习，我不仅巩固了 Java、Spring Boot、React 等技术的实际运用能力，更重要的是学会了如何从业务需求出发进行技术选型和系统设计，体会了软件工程规范（版本控制、测试驱动、CI/CD）在真实项目中的价值。这次实习让我对全栈开发有了更深入的理解，也为今后从事软件开发工作积累了宝贵的实践经验。
