# 客户数据管理系统 — 项目计划书（from: deepseek-v4-flash）

## 1. 项目概述

| 项目 | 内容 |
|------|------|
| 项目名称 | 博易通客户数据管理系统 |
| 开发目的 | 为安阳博易通网络有限公司搭建客户信息管理平台，替代 Excel 手工管理方式 |
| 实习方向 | 全栈 Web 开发 + 数据处理与可视化 |
| 后端技术栈 | Spring Boot 3.x + Spring Data JPA + PostgreSQL + Apache POI + Spring Security + JWT |
| 前端技术栈 | React (Vite) + Tailwind CSS + Chart.js |
| API 风格 | RESTful API + SpringDoc OpenAPI (Swagger) |
| 测试 | JUnit 5 + Mockito（后端） |
| CI/CD | GitHub Actions |
| 部署方案 | Railway 免费部署 |

## 2. 功能需求

### MVP（最小可行产品）
- [x] Excel 数据批量导入
- [x] 客户信息展示列表（分页/排序）
- [x] 多条件筛选（城市、区域、行业/板块、面积范围、销售员）
- [x] 关键词搜索（地址、备注）
- [x] 数据统计仪表盘
  - 各城市客户数量分布
  - 各行业/板块占比
  - 各销售员业绩统计
  - 面积分布分析
- [x] 数据可视化图表

### 新增功能（认证 + 审核）
- [x] **用户认证** — JWT 登录（Spring Security），管理员/业务员双角色
- [x] **上传审核流程** — 业务员上传 → 待审核 → 管理员通过/拒绝 → 数据并入主表

### 后续展望
- [ ] 客户信息增删改查（CRUD）
- [ ] 数据导出（Excel/PDF 报告）
- [ ] 销售团队多用户登录
- [ ] 跟进记录功能

## 3. 技术架构

```
┌──────────────────────────────────────────┐
│           前端 (React + Vite)             │
│   AuthContext + pages/ → components/     │
│   Tailwind CSS + Chart.js                │
├──────────────────────────────────────────┤
│         RESTful API (JSON + JWT)         │
├──────────────────────────────────────────┤
│      后端 Spring Boot 3.x (Java 17)       │
│  Controller → Service → Repository (JPA) │
│  Spring Security + JWT 认证              │
│  Apache POI (Excel 解析)                  │
├──────────────────────────────────────────┤
│             PostgreSQL                    │
└──────────────────────────────────────────┘
```

### API 接口设计

```
GET    /api/customers          # 客户列表（分页 + 筛选 + 搜索）
GET    /api/customers/{id}     # 客户详情
POST   /api/import             # 导入 Excel
GET    /api/stats/cities       # 城市分布统计
GET    /api/stats/categories   # 行业分布统计
GET    /api/stats/sales        # 销售员统计
GET    /api/stats/area         # 面积分布统计
```

### 数据流

```
用户操作 → React state 更新 → axios 调 API → Spring Boot → JPA → PostgreSQL
                                                                     ↓
用户看到新页面 ← React re-render ← JSON 响应 ← Controller ← Service
```

## 4. 实施路线图

### Phase 1：项目初始化
1. 创建 Spring Boot 项目骨架（Maven + pom.xml）
2. 配置 application.yml（数据库、跨域等）
3. 设计 Entity 数据模型（Customer）
4. 编写 Excel 数据导入功能（Apache POI）
5. 将柳州 & 鄂尔多斯数据导入 PostgreSQL
6. 创建 React 项目 (Vite)
7. 配置 Tailwind CSS
8. 搭建 Layout + 路由

### Phase 2：后端 API 开发
9. 客户列表接口（分页 + 多条件筛选 + 搜索）
10. 客户详情接口
11. 统计接口（城市/行业/销售员/面积分布）
12. 编写单元测试（JUnit 5 + Mockito）

### Phase 3：前端页面开发
13. 客户列表页（FilterBar + Table + Pagination）
14. 客户详情页
15. 仪表盘页（StatCard + BarChart + PieChart）

### Phase 4：CI/CD + 部署
16. 配置 GitHub Actions CI 流水线
17. 部署到 Railway
18. 配置 PostgreSQL 生产数据库
19. 导入生产数据
20. 端到端验证

### Phase 5：报告撰写
21. 撰写实习报告
22. 整理开发文档
23. 准备答辩材料
